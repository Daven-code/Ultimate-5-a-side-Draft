## Important change

The online version is now different to the local version:

- Host creates a room and gets a waiting/lobby screen.
- Other users join the room with their own name.
- The host continues to setup and types the same user names into the game setup boxes.
- In Ultimate Draft 5-a-side, only the person whose name matches the current turn can press Pick/Accept/Decline.
- Other devices stay locked and update live.

Bid mode is still best treated as host-assisted for now; the main turn-by-turn online control is for Ultimate Draft 5-a-side.
