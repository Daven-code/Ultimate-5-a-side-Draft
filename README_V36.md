# Ultimate 5-a-side Draft v36

Based on v35. This update improves the final results layout for 4-player local/online games.

Updates:
- The finished results page now uses a roomier two-column layout on desktop instead of squeezing four result cards into one row.
- Cards are wider, player rows have more breathing room, and pitches have a little more height.
- On tablets/mobiles, results stack into one column.
- Gameplay behaviour from v35 is unchanged.
- index.html references `app.js?v=36` for cache busting.

Install:
1. Replace only `index.html` and `app.js`.
2. Do not replace `styles.css` or `players.json`.
3. Commit and push.
4. Hard refresh with Ctrl+F5.
