# Ultimate 5-a-side Draft v40

Based on v39. This update improves the ONLINE blind bidding page UI only.

Changes:
- Clearer online blind bidding layout.
- Step-by-step instructions for the current user.
- More prominent player card.
- Larger private bid input and submit button.
- Clear status cards showing who is waiting, submitted, or not eligible.
- Progress bar showing how many eligible players have submitted.
- Improved bids revealed panel.
- Blind bid behaviour and local bid behaviour are unchanged from v39.

Install:
1. Replace only `index.html` and `app.js`.
2. Do not replace `styles.css` or `players.json`.
3. Commit and push.
4. Hard refresh with Ctrl+F5.
5. Create a new online room when testing.
