# Ultimate 5-a-side Draft v44 - Online bid skip lives

Based on the uploaded v43 files.

Changes in v44:
- In ONLINE blind bidding, a £0m bid now uses one of that user's 3 skips.
- If an online user has used all 3 skips, they must bid above £0m.
- The online bid screen now shows each user's skips left.
- When bids are revealed, users who bid £0m are listed as losing one skip.
- Local v43 skip lives are retained unchanged.

Install:
1. Replace only `index.html` and `app.js` in your GitHub repo.
2. Do not replace `styles.css` or `players.json`.
3. Commit and push.
4. Hard refresh with Ctrl+F5.
5. Create a new online room to test.
