## Changes in v20

- The host lobby now contains the game start options directly.
- You no longer need to go back to the normal setup page to start an online game.
- Joined player names are used automatically when the host clicks **Start online game**.
- No more User 1 / User 2 issue for online games, provided players joined with their names.
- Ultimate Draft 5-a-side remains turn-based: only the current named player can Pick/Accept/Decline.
- Bid mode is still host-assisted for now, but devices still update live.
